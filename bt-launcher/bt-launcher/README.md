# Based Trenches Launcher — Chrome Extension

Launch memecoins on Base Chain directly from your browser. Tokens appear on basedtrenches.fun automatically.

## Setup (Developer Mode)

1. Download and unzip this folder
2. Open Chrome and go to: chrome://extensions
3. Toggle ON "Developer mode" (top right)
4. Click "Load unpacked"
5. Select this folder (bt-launcher)
6. The 🪖 icon will appear in your toolbar

## Features
- Full token launch form (name, symbol, description, image, banner, socials)
- Live token preview as you type
- Connects to MetaMask, Coinbase Wallet, Rabby
- Auto-switches to Base network
- Images uploaded to IPFS
- Token appears on basedtrenches.fun within 30 seconds
- Live Trenches tab shows all active tokens

## Publishing to Chrome Web Store
1. Go to https://chrome.google.com/webstore/devconsole
2. Pay $5 one-time developer fee
3. Click "New Item" → Upload a zip of this folder
4. Fill in store listing details
5. Submit for review (usually 1-3 days)
